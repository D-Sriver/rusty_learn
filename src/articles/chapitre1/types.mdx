En Rust, il existe plusieurs types de base. Pas de panique, c'est comme dans la plupart des langages, mais avec quelques subtilités !

## Les principaux types

- `i32`, `u32`, `i64`, etc. : entiers signés et non signés
- `f32`, `f64` : nombres à virgule flottante
- `bool` : vrai ou faux
- `char` : un caractère Unicode (pas juste une lettre !)
- `String` et `&str` : chaînes de caractères
index
> **Astuce de Rusty**  
> En Rust, tu dois préciser le type si le compilateur ne peut pas le deviner. Mais souvent, il est malin et devine tout seul !

## Exemples concrets

```rust
let age: u32 = 30;
let temperature: f64 = 23.5;
let actif: bool = true;
let lettre: char = 'R';
let nom: &str = "Rusty";
```

## Parallèle Web


En JS, tu peux écrire :

```js
let age = 30;
let nom = "Rusty";
```

En Rust, c'est pareil, mais tu peux (et parfois dois) préciser le type !
