# Introduction à Rust

Bienvenue dans ce cours interactif sur **Rust** !

## Pourquoi apprendre Rust ?

- Sécurité de la mémoire
- Performance
- Concurrence sans crainte
- Large communauté
- Documentation exemplaire

> "Rust vous permet d'écrire des logiciels rapides et fiables."

## Premiers pas

```rust
fn main() {
    println!("Hello, world!");
}
```

Ce code affiche `Hello, world!` dans la console.

---

## Syntaxe de base

### Variables

```rust
let mut compteur = 0;
compteur += 1;
```

### Conditions

```rust
if compteur > 0 {
    println!("Compteur positif");
} else {
    println!("Compteur nul ou négatif");
}
```

## Comparaison avec d'autres langages

| Concept         | Rust         | Python      | C++         |
|-----------------|-------------|-------------|-------------|
| Mutabilité      | `let mut`   | variable    | variable    |
| Sécurité mémoire| Oui         | Non         | Non         |
| Gestion mémoire | Automatique | Automatique | Manuelle    |
