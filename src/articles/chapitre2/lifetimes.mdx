# Lifetimes

Les lifetimes garantissent que les références restent valides aussi longtemps qu'elles sont utilisées.

```rust
fn plus_longue<'a>(x: &'a str, y: &'a str) -> &'a str {
    if x.len() > y.len() {
        x
    } else {
        y
    }
}
```

Les annotations de lifetime sont souvent nécessaires pour les fonctions qui retournent des références.
