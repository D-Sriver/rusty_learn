# Borrowing

Le borrowing permet de prêter une valeur sans en transférer l'ownership.

- `&T` : référence immuable
- `&mut T` : référence mutable

```rust
fn afficher(message: &String) {
    println!("{}", message);
}

let texte = String::from("Rusty!");
afficher(&texte); // texte n'est pas déplacé
```
